# Kobo Manager

## Descripción

Aplicación de escritorio desarrollada en Java para explorar, analizar y exportar datos almacenados por dispositivos Kobo eReaders.

El objetivo del proyecto es crear una herramienta que permita consultar información de lectura, organizar los datos obtenidos del dispositivo y generar exportaciones que faciliten su análisis posterior mediante herramientas de visualización.

## Estado del proyecto

🚧 En desarrollo

Actualmente el proyecto se encuentra en una fase inicial, trabajando en la estructura de la aplicación, diseño de la arquitectura y desarrollo de las funcionalidades principales.

## Funcionalidades previstas

- Importación y consulta de datos procedentes de Kobo eReaders.
- Exploración de información de libros y estadísticas de lectura.
- Gestión y almacenamiento de datos mediante base de datos SQLite.
- Filtrado y organización de la información.
- Exportación de datos para su análisis.
- Integración con herramientas de visualización como Power BI.
- Interfaz gráfica de escritorio intuitiva.

## Tecnologías utilizadas

- Java
- Java Swing
- JDBC
- SQLite
- Power BI
- Git / GitHub
